# Rumor Propagation Engine (C++)

## 📌 Overview

This project simulates how a rumor spreads among people in a network.
Each person is connected to others, and the rumor spreads step by step from one person to their neighbors.

---

## 🎯 Objective

To demonstrate rumor spreading using:

* Object-Oriented Programming (OOP)
* Graph representation (Adjacency List)

---

## 🧠 Concepts Used

* Classes and Objects
* Vectors (STL)
* Graph (Nodes and Edges)
* Loops and Simulation

---

## ⚙️ How It Works

* Each person is represented as a node.
* Connections between people are stored using vectors.
* Initially, one person starts the rumor.
* In each step, the rumor spreads to connected neighbors.

---

## ▶️ Execution Steps

1. Create people (nodes)
2. Add connections between them
3. Start rumor from one person
4. Run simulation step by step
5. Display output

---

## 💻 Sample Output

Step 0: 0
Step 1: 0 1
Step 2: 0 1 2
Step 3: 0 1 2 3

---

## 🚀 Applications

* Social media information spread
* Viral marketing
* Basic network simulations

---

## 📌 Conclusion

This project shows how rumors spread in a network using simple C++ concepts and graph structure.
